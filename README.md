# IaC Generator
A simple IaC generator that produces portable IaC code based on the detected infrastructure configuration.

## Usage
1. Create an instance of `InfrastructureConfig` with the desired platform and resources.
2. Call `generate_iac_code` to generate the IaC code.
3. Review and modify the generated IaC code using `review_and_modify_iac_code`.

## Testing
Run `python -m pytest` to execute the tests.
